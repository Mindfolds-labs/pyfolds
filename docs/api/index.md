# API Documentation

## Objetivo
Definir o ponto de entrada para documentação de API.

## Escopo
Página placeholder com estratégia evolutiva para autodoc/autosummary.

## Definições/Termos
- **Autodoc:** geração automática de docs a partir de docstrings.

## Conteúdo técnico
Estratégia prevista:
1. Consolidar docstrings por módulo em `src/pyfolds`.
2. Habilitar `sphinx.ext.autodoc` e `sphinx.ext.autosummary`.
3. Publicar referência por domínio (`core`, `advanced`, `network`, etc.).
