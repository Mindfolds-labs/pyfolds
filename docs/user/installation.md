# Installation

## Objetivo
Descrever como instalar o PyFolds para uso local.

## Escopo
Ambiente Python e instalação via pip.

## Definições/Termos
- **Ambiente virtual:** isolamento de dependências do projeto.

## Conteúdo técnico
1. Crie e ative um ambiente virtual.
2. Instale o pacote:

```bash
pip install pyfolds
```

3. Para documentação local:

```bash
pip install -r requirements-docs.txt
```
