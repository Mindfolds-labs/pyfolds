# Installation

## Requisitos

- Python 3.8+
- PyTorch compatível

## Instalação via pip

```bash
pip install pyfolds
```

## Instalação para desenvolvimento

```bash
git clone https://github.com/Mindfolds-labs/pyfolds.git
cd pyfolds
python -m venv .venv
source .venv/bin/activate
pip install -e .[dev]
```
