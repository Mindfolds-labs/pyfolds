# Context
A literatura aponta delays regionais no processamento de envelope.

# Decision
Adicionar `latency_kernel` espacial opcional aplicado à latência wave.

# Rationale
Aumenta realismo sem alterar baseline padrão.

# Tradeoffs
Kernel isotrópico simplifica demais estrutura anatômica.
