# Contributing

## Objetivo
Definir fluxo de contribuição para mudanças em código e documentação.

## Escopo
Branching, commits, testes e revisão.

## Definições/Termos
- **PR:** *Pull Request* para revisão e integração.

## Conteúdo técnico
1. Crie branch por issue.
2. Execute checks locais.
3. Atualize documentação quando aplicável.
4. Abra PR com evidências.
