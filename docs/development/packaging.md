# Packaging

## Objetivo
Orientar build e distribuição do pacote Python.

## Escopo
Build local, metadados e artefatos.

## Definições/Termos
- **Wheel:** distribuição binária padrão do ecossistema Python.

## Conteúdo técnico
```bash
python -m build
pip install dist/*.whl
```
