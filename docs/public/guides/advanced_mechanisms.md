# Guide — Advanced Mechanisms

Mecanismos avançados disponíveis:

- STDP
- Refratário (absoluto/relativo)
- Adaptação (SFA)
- Inibição lateral/feedback
- Backpropagação dendrítica
- Dinâmica de curto prazo (facilitação/depressão)

Use esses módulos quando precisar de maior realismo biológico ou dinâmica temporal mais rica.
