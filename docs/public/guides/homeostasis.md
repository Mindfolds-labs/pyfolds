# Guide — Homeostasis

Homeostase regula excitabilidade para manter taxa de disparo próxima da meta.

Principais variáveis:

- `theta`: limiar somático adaptativo.
- `r_hat`: estimativa de taxa média.
- `target_spike_rate`: alvo de atividade.

Efeito prático: evita neurônio morto (subatividade persistente) e saturação excessiva.
